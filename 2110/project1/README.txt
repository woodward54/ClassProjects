Project 1: Nurse Scheduler

This program takes a series of Nurses and their weekly schedules. It stores their schedules in a 3d bitvector and performs various set operations to achieve required outputs.
It then finds the minimal nurse and shift schedule.

Final Grade: 110/100

Compiling
- compiled using c++ 11
- "g++ -std=c++11 main.cpp"

Running
- "./a.out" to run the program

File Input
- The program will ask for the file name, input the exact name for example "input3.txt"
- Make sure the input file is in the same directory as main.cpp
