Project 2: Outbreak Simulator

This program takes an input file with a starting matrix of who is in what stage of the diese. It then visually simulates the outbreak spread based on ruled provided.

Compiling
- compiled using c++ 11
- "g++ -std=c++11 *.cpp"

Running
- "./a.out" to run the program

File Input
- The program will ask for the file name, input the exact name for example "input3.txt"
- Make sure the input file is in the same directory as main.cpp
- input1.txt and input2.txt and NOT bonus files, bonusinput1.txt is a bonus file
